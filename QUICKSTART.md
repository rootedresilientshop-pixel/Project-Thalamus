# Thalamus Quickstart Guide

## Installation

### Step 1: Download from itch.io
1. Visit: https://dreamcraftstudio.itch.io/thalamus
2. Download the latest ZIP file (v1.0.0-GOLD or newer)
3. Extract to your preferred location

### Step 2: Launch
- **Windows**: Double-click `Thalamus.UI.exe`
- **macOS/Linux**: Run the appropriate executable for your platform

That's it. Thalamus runs entirely locally. No account. No sign-up. No cloud dependency.

---

## Node Nexus: Your Module Sidebar

When you launch Thalamus, the left sidebar displays **Node Nexus**—your gateway to the module ecosystem.

### Loading the 5 Seed Templates

The GOLD release includes 5 pre-built seed templates to jumpstart your workflow:

1. **Data Pipeline** — Transform raw data into structured insights
2. **Logic Orchestrator** — Chain complex workflows with visual logic gates
3. **Hardware Bridge Gateway** — Prepare your environment for BridgeMod integration
4. **Security Audit Framework** — Set up local governance tracking (Kanon-ready)
5. **Module Template Scaffold** — Start building your own modules

**To load a template:**
1. Open Node Nexus sidebar
2. Click "Seed Templates" → Select your template
3. Click "Load" → Template deploys to your workspace

All templates are 100% local. Modify them, share them, or keep them private.

## The Module Ecosystem

### The Core is Your Chassis

Thalamus Core is the engine. It gives you:
- Local logic execution
- Module hosting framework
- Data management (your .thlm files)
- Visual workflow editor

### Kanon is Your Security

[Kanon Sentry Kernel](https://github.com/dreamcraftstudio/kanon-sentry-kernel) adds sovereign governance:
- Cryptographic audit logging
- Permission management
- Transparent governance framework
- Integrates with Thalamus Core

**When to use Kanon**: When you need institutional-grade security, audit trails, or compliant deployments.

### BridgeMod is Your Hands

[BridgeMod Hardware Bridge](https://github.com/dreamcraftstudio/bridgemod-hardware-bridge) extends Thalamus to physical devices:
- Raspberry Pi orchestration
- Arduino bridging
- IoT sensor integration
- Local hardware abstraction

**When to use BridgeMod**: When you want to control hardware (sensors, relays, microcontrollers) from Thalamus without cloud APIs.

## Node Nexus: Share & Monetize

### Upload Your Modules
1. Build a module in Thalamus (or contribute to existing ones)
2. Open Node Nexus → "My Modules"
3. Click "Publish to Marketplace"
4. Set your price (or free)
5. Done. 100% of proceeds go to you.

### Download Community Modules
1. Open Node Nexus → "Community Marketplace"
2. Browse, preview, or install modules from creators worldwide
3. All modules run locally on your machine
4. Support creators directly—no middleman.

## Your .thlm Files

Every workflow, module, and configuration you create is saved as a `.thlm` file.

- **100% Ownership**: You own all .thlm files. Export them, back them up, share them.
- **No Vendor Lock-in**: If you stop using Thalamus, your .thlm files are still yours.
- **Portable**: Move .thlm files between machines freely.
